# BmpFadeTop
Script implementing a "fading" efect for a given number of pixels of BMP.

In order to use the program you can use

    ./program <your_bmp_image.bmp> <height>


Where the height should be a number of pixels you want to fade effect to have.

Example of effects:

after:

./program Tsunami.bmp 200
